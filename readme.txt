1. O que é o projeto?
    O Governo do Estado do Maranhão, por meio da SECTI em parceria com a FAPEMA, está promovendo um programa de formação de jovens e adultos na área de tecnologia. O programa segue estas fases: 
    •	Liberação do Edital ao Público 
    •	Período de Inscrição 
    •	Seleção na primeira fase via teste de nivelamento 
    •	Seleção na segunda fase via teste técnico 
    •	Processo de Matrícula no Programa 
    Para gerenciar possíveis 10.000 inscritos por ano, é necessária uma ferramenta robusta e planejada. 
    Pensando nisso, esta ferramenta foi desenvolvida com o objetivo de gerenciar as inscrições de tais usuários, bem como facilitar o acesso à formação promovida pela SECTI e FAPEMA. 

2. Como rodar localmente?
    Para rodar o projeto basta apenas usar uma extensão do vs code que serve a aplicação no navegador local refletindo as alterações em tempo real,
    ou abrir estaticamente clicando duas vezes e escolhendo um navegador.
3. Tecnologias utilizadas.
    Para o desenvolvimento da ferramenta apresentada, foram utilizados HTML como linguagem de marcação, CSS para estilização da página, JavaScript para implementação de interatividade e funcionalidades dinâmicas, e GitHub para versionamento e controle do código.
 

4. Principais funcionalidades.
    1.	Campos de formulário com labels para guiar o preenchimento pelo usuário; 
    2.	Formatação automática do campo a ser preenchido; 
    3.	Combobox para seleção de gênero, agilizando o preenchimento; 
    4.	Opção funcional para envio de arquivo; 
    5.	Menu interativo funcional para escolha da trilha desejada. 
    6.	Armazenamento temporário de dados
    7.	Sistema de login responsivo
    8.	Validação em Tempo Real: Verificação instantânea de formato de e-mail
    9.	Armazenamento Local: Persistência de dados com LocalStorage
    10.	Responsividade: Adaptação a diferentes tamanhos de tela
    11.	Sistema de Login Seguro: Validação de credenciais
    12.	Feedback Visual: Mensagens de erro contextualizadas
    13.	Componentes Modernos: Combobox, upload de arquivos e campos específicos
